## ✨ [High Priority] 유료 Mockup 미리보기 템플릿 디자인 시스템 구축

CEO의 지시와 Business팀의 '개인 자산화' 전략(Tiered Pricing)을 반영하여, 사용자가 무료 경험 후 즉각적으로 소유하고 싶게 만드는 **고화질 비주얼 컴포넌트**를 제작합니다. 가장 노출 빈도가 높고 구매 욕구를 자극하는 플랫폼에 맞춰 2가지 핵심 Mockup 시안과 통합 디자인 시스템(DSD)을 제안합니다.

### 1. 🖼️ 핵심 산출물: Instagram Story / Link-in-Bio 미리보기 템플릿 (Mockup Preview)

이 컴포넌트는 사용자가 웹사이트를 떠나지 않고도, **"이거 내 거다!"**라는 소유감을 느끼게 하는 것이 목표입니다.

| 항목 | 상세 스펙 및 디자인 가이드라인 | 적용 요소 및 코딩 지침 |
| :--- | :--- | :--- |
| **목적** | 이름의 '공식적인' 증명서/소셜 자산으로서의 시각화 (구매 전환 유도) | CTA를 중심으로 배치하여 클릭 흐름을 강제합니다. |
| **규격** | 9:16 비율 (Story 표준) / 이미지 폭 기준 약 1080px, 높이 1920px | 배경에 약간의 그림자(Shadow) 효과를 주어 '실물' 같은 입체감을 부여합니다. |
| **디자인 컨셉** | **[Premium Credibility]**: 미니멀리즘 기반의 고급스러움 + 공신력 있는 배지/인증서 느낌 혼합. | 톤앤매너는 신뢰감 있는 네이비 계열과 골드 포인트를 활용합니다. |
| **레이아웃 구조 (Top-Down)** | **① 상단**: 'The Official Name Asset' 문구 및 이름의 의미 요약 배지<br>**② 중앙(핵심)**: [사용자명] + [별칭/직업]을 고급 폰트로 크게 배치. (가장 큰 시각적 충격 지점)<br>**③ 하단**: 'Complete Your Story', 'Mockup 미리보기' CTA 버튼 및 구매 정보 | **배경색:** `#F5F7FA` (미세한 그레이 베이스).<br>**텍스트 색상:** `#1E2A40` (짙은 네이비).<br>**액센트 컬러:** `#D4AF37` (골드 포인트) |
| **핵심 인터랙션** | 사용자가 이 Story를 공유 버튼을 누르면, 자동으로 구매 페이지(Tier 2/3 랜딩)로 연결되는 `Deep Link`가 활성화되어야 합니다. | CTA 버튼 영역에 미세한 애니메이션 (`Pulse Effect`)를 적용하여 시선을 유도합니다. |

### 2. 🌐 확장 컴포넌트: 디지털 명함/프로필 카드 (Link-in-Bio용)

사용자가 링크 트리거로 활용할 수 있는, 웹사이트 외 공간에서의 '나'의 정체성을 보여주는 자산입니다.

| 항목 | 상세 스펙 및 디자인 가이드라인 | 적용 요소 및 코딩 지침 |
| :--- | :--- | :--- |
| **규격** | 1:1 비율 (Instagram/Link-in-Bio 표준) / 이미지 폭 기준 약 1080px, 높이 1080px | 모서리(Corner Radius)를 적절히 주어 딱딱한 느낌을 완화합니다. |
| **레이아웃 구조** | **① 상단**: 프로필 사진 (Placeholder).<br>**② 중앙**: 이름과 핵심 별칭 (가장 강조될 영역)<br>**③ 하단**: `havenames.com/user-name` 형식의 전용 URL 및 "Certified Name Asset" 배지 | **텍스트 크기:** 이름을 가장 크게, 브랜드 컬러를 활용하여 시각적 계층을 명확히 합니다.<br>**배치 원칙:** 중앙 정렬(Center Alignment)으로 안정감 부여. |
| **활용 가이드** | 이 이미지를 통해 사용자는 "나는 이름 서비스로 공인된 사람이다"라는 메시지(Social Proof)를 간접적으로 전달하게 됩니다. | 이미지 하단에 `© QARAH, Asset Certified` 워터마크를 작게 삽입하여 소유권을 명확히 합니다. |

### 💡 디자인 시스템 요약 (Design System Definition)

| 요소 | 상세 스펙 | 활용 목적 및 지침 |
| :--- | :--- | :--- |
| **주요 컬러 (Primary)** | `#1E2A40` (딥 네이비) - 깊이감, 신뢰성 | 주요 텍스트, 배경 대비, 중요 섹션 분리. |
| **액센트 컬러 (Accent)** | `#D4AF37` (골드/브론즈) - 고급스러움, 프리미엄 | CTA 버튼 배경, 인증 마크, 하이라이팅 포인트에만 제한적으로 사용. 과용 금지. |
| **배경색 (Background)** | `#F5F7FA` (화이트 그레이) - 가독성, 여백감 확보 | 전체적인 콘텐츠 영역의 기본 바탕색으로 활용하여 산뜻함을 유지합니다. |
| **폰트 (Typography)** | 1. Headline: **Pretendard Bold** (혹은 Noto Sans KR Black 등 강한 인상) / 2. Body: Pretendard Regular | 제목에는 힘을 주어 '자산화'의 무게감을, 본문에는 가독성을 최우선으로 합니다. |
| **컴포넌트 패턴** | **[Asset Badge]**: 직사각형 형태에 모서리 라운딩 처리된 배지 형태로 제작 (예: `Certified Name Asset`). 이는 소유권의 시각적 증명 역할을 합니다. |
