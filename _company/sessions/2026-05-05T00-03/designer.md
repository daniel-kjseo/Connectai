## ✨ 최종 결과 페이지 (The Assetization Funnel) 디자인 컴포넌트 스펙

### 1. 핵심 목표: 감성적 전환 유도
*   **전환 버튼의 목적:** 구매 요청 $\rightarrow$ **가치 보존/소유권 확보 과정 참여 유도.**
*   **핵심 원리:** 사용자가 '무료로 보는 것'과 '완벽히 소유하는 자산' 사이의 간극(Gap)을 명확하게 인지하도록 설계합니다.

### 2. 컴포넌트 A: 고화질 Mockup 미리보기 영역 (The Asset Showcase)
사용자에게 가장 큰 시각적 만족감을 제공하는 곳입니다.

| 요소 | 스펙/규칙 | 디자인 지침 (Actionable Spec) |
| :--- | :--- | :--- |
| **배치** | 페이지의 최상단 중앙 (Hero Section). 이름 추천 결과와 분리하여 별도의 '소유 자산' 영역으로 격리. | 전체 섹션 배경색: `#F9FAFB` (약간의 대비로 중요도 강조) |
| **Mockup 뷰** | 모바일/웹 명함 Mockup을 나란히 배치하거나, 인터랙티브 슬라이드 형태로 구성. | **효과:** 물리적 오브제(실물 자산)처럼 보이도록 은은한 그림자 (`box-shadow: 0 15px 30px rgba(0,0,0,0.1);`)와 깊이감(Depth)을 부여합니다. |
| **시각화 포인트** | 이름 데이터가 들어간 자산의 '개인 맞춤성'을 극대화해야 합니다. (예: 배경 이미지에 사용자의 가상 직업군이나 취미 관련 요소를 은근히 녹여냄). | **하이라이팅:** 가장 핵심적인 개인 정보(사용자 이름, 추천 의미 등)에는 미세한 골드/딥 블루 색상의 하이라이트 효과를 적용하여 고급스러움을 더합니다. |
| **UX 요소** | '3D 회전 보기' 또는 '실물로 보는 느낌'의 인터랙션 애니메이션을 필수적으로 포함해야 합니다. |

### 3. 컴포넌트 B: 전환 유도 CTA 블록 (The Ownership Gateway)
사용자가 자산에 매료된 직후, 망설임을 구매 동기로 바꾸는 구간입니다.

| 요소 | 스펙/규칙 | 디자인 지침 (Actionable Spec) |
| :--- | :--- | :--- |
| **섹션 제목** | 감성적 질문 던지기: "이 소중한 가치를 영원히 간직하시겠어요?" 또는 "당신의 이야기를 완성할 시간입니다." | 폰트: Heading Font (세리프 계열) 사용. 색상: `#333` (진중함). |
| **가치 설명(Microcopy)** | 무료 Mockup의 한계점과 유료 자산의 가치를 대비하여 명시해야 합니다. (예: *무료 버전은 '미리보기'에 불과합니다.* vs *프리미엄 패키지는 영구 소유권 및 공식 활용 라이선스가 부여됩니다.*) | 목록 형태로 3가지 핵심 차이점(소유권, 확장성, 고해상도 등)을 나열하고 체크 표시(`✓`)를 사용합니다. |
| **CTA 버튼 (Primary)** | 구매 유도를 감성적으로 치환해야 합니다. | **텍스트 옵션 제시 (A/B 테스트용):**<br>1. 💖 나의 추억 영구 보존하기 ($XX)<br>2. ✨ 공식 자산으로 완성하기<br>3. 👑 프리미엄 라이선스로 소유하기<br>**색상:** 브랜드의 메인 포인트 컬러(딥 블루 계열)를 사용하되, 채도를 높여 '클릭해야 할' 느낌을 강조합니다. |
| **CTA 버튼 (Secondary)** | 낮은 가격대의 진입 장벽을 제공하는 옵션. | **예시:** "일단 가이드북만 받아보기" ($9)<br>**배치:** Primary CTA보다 크기를 작게, 배경색은 투명하게 처리하여 보조적인 선택지로 만듭니다. |

### 4. 디자인 시스템 규칙 (Global Rule Set)
| 영역 | 스펙 | 코드/지침 | 활용 목적 |
| :--- | :--- | :--- | :--- |
| **메인 타이포** | 제목 및 강조: 세리프 계열 (클래식, 고급스러움). | 예: Noto Serif KR / Playfair Display | 소유감, 역사성 부여. |
| **보조 타이포** | 본문 및 정보 전달: 산세리프 계열 (가독성 최우선). | 예: Pretendard / Noto Sans KR | 가독성과 깔끔함 유지. |
| **컬러 팔레트** | 딥 블루 & 골드 조합으로 '고급 자산' 느낌 강조. | **Primary:** `#1A237E` (Deep Indigo)<br>**Secondary:** `#FFD700` (Soft Gold Accent)<br>**Background:** `#F9FAFB` (Clean White) | 프리미엄하고 신뢰감 있는 브랜드 톤 유지. |
