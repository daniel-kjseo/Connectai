## ✨ 릴스/피드 콘텐츠 플랜: 서사 기반 참여형 바이럴 전략

새롭게 정의된 '서사(Narrative)' 중심의 가치를 활용하여, 타겟 청중이 쉽게 따라 하며 재미를 느낄 수 있는 **참여 유도형(Participation-driven)** 릴스 콘텐츠 3개를 기획했습니다. 모든 콘셉트는 QARAH 서비스 사용을 자연스럽게 유도하는 흐름으로 설계되었습니다.

### 🎬 Reel Concept 1: [🔥 바이럴/호기심 자극] "당신의 이름은 사실 비밀코드다?" (The Name Decoder)

**목표:** 초기 인지도를 높이고, '이름'에 대한 근본적인 호기심을 유발하여 서비스에 관심을 갖게 한다.
**포맷:** 빠른 템포의 텍스트/영상 전환, 트렌딩 배경음악 활용 (사운드 원탑재 필수).

| 요소 | 상세 내용 및 스크립트 가이드 | 실행 가능한 액션 아이템 |
| :--- | :--- | :--- |
| **훅(Hook)** | (0-3초) "당신의 이름, 그냥 듣는 대로 부르는 게 아닙니다." <br> *(시각: 자막 강조. 긴장감 있는 사운드 효과)* | 🚨 시선을 잡아끄는 충격적인 질문으로 시작. |
| **보이스오버** | (4-10초) "이름은 단순히 아빠가 지어준 단어가 아니에요. 당신의 *본질*을 담고, 살아온 *서사*를 품은 비밀코드와 같습니다." <br> (11-20초) "그 코드가 무엇인지 궁금하다면? QARAH에서 나만의 서사를 디코딩해보세요." | '비밀'과 '디코딩'이라는 키워드를 반복하여 사용. |
| **자막/화면 구성** | 🔹 (Before): [일반적인 이름] $\rightarrow$ ❌ 의미 없음<br> 🔹 (After): QARAH 디코드 $\rightarrow$ ✅ 서사 + 잠재력 + 나만의 가치 | 명확한 대비(Contrast)를 통해 문제 제기 및 해결책 제시. |
| **캡션** | ✨ 혹시 당신의 이름에 숨겨진 비밀이 있다고 생각해 본 적 있나요? 우리의 이름은 단순한 라벨이 아니라, 우리가 알지 못하는 잠재력을 담고 있는 지도 같은 거예요. QARAH에서 나만의 코드를 찾아보세요! 🗝️ (링크) | 질문을 던져 댓글 참여를 유도합니다. |
| **해시태그** | #이름의비밀코드 #서사찾기 #나만의이야기 #QARAH #명리학 #운세 #자기계발 | *브랜드 해시태그* + *광범위 키워드* + *니치(심화) 키워드* 조합. |
| **게시 시간** | 화요일 오후 6:30 (퇴근 후 정보 탐색 및 소비가 활발한 시간대) |
| **후속 스토리 아이디어** | "🤔 여러분의 이름에 숨겨진 비밀은 무엇일까요? 친구를 태그해서 같이 디코딩해봐요!" (댓글 유도 스티커 활용) |

### 🎬 Reel Concept 2: [🎓 참여형/교육] "3단계! 나만의 이름 서사 만드는 법" (The Workshop Guide)

**목표:** 서비스 사용 과정을 교육적으로 제시하여, 무료 사용자에게 '다음 단계 구매'의 당위성을 부여한다.
**포맷:** 가이드 형식의 Voiceover + 화면 녹화(Mockup 시연) 조합. (가장 중요한 전환 콘텐츠).

| 요소 | 상세 내용 및 스크립트 가이드 | 실행 가능한 액션 아이템 |
| :--- | :--- | :--- |
| **훅(Hook)** | (0-3초) "내 이름, 그냥 아는 것에서 그치면 안 돼요. 스토리로 만들어야 합니다." <br> *(시각: 'HOW TO' 텍스트 크게 강조)* | 구체적인 방법을 제시하며 실질적 도움을 준다는 인상을 심어줌. |
| **보이스오버** | (4-15초) "준비물은 딱 3가지. ① 나만의 이름 ② 나의 가장 중요한 가치 ③ 이 이름을 어떻게 쓰고 싶은가? QARAH는 이 세 가지를 연결해 스토리로 완성시켜줘요." <br> (16-25초) *[화면 시연]* -> Mockup 미리보기 → "이게 바로 소유권이에요!" | **명확한 단계 제시**와 함께, 유료 기능(Mockup/Assetization)을 자연스럽게 노출. |
| **자막/화면 구성** | 🟢 Step 1: 기본 의미 찾기 (무료)<br>🟡 Step 2: 가치 연결하기 (유료 전환 지점 강조)<br>🔴 Step 3: 자산으로 완성하기 (최종 구매 CTA) | 각 단계마다 '✨' 이모티콘과 함께 진행 상황을 보여주어 흥미를 유지. |
| **캡션** | 📝 나만의 스토리를 만드는 건 생각보다 쉬워요! QARAH가 알려주는 이름 서사화 워크숍 가이드입니다. 단순히 이름을 아는 것을 넘어, '나의 자산'으로 만들 시간이에요. 프로필 링크에서 지금 바로 체험해보세요! 👇 | 실용적인 정보를 제공하며 CTA를 강력하게 배치합니다. |
| **해시태그** | #이름스토리텔링 #라이프코칭 #자기계발꿀팁 #QARAH가이드 #명함디자인 #브랜딩명함 | '정보 습득'과 '실용성'에 초점을 맞춘 키워드 사용. |
| **게시 시간** | 목요일 오전 10:00 (업무 중 잠시 쉬는 시간에 교육적 콘텐츠를 소비하는 경향이 높음) |
| **후속 스토리 아이디어** | "🎬 릴스 보셨나요? 3단계 과정 중에 가장 어려웠던 단계는 무엇인가요? 댓글로 알려주시면 다음 스토리에 QARAH가 코칭해 드릴게요!" (댓글 유도 및 참여 심화) |

### 🎬 Reel Concept 3: [❤️ 감성/관계 기반] "우리의 이름은 어떻게 연결될까?" (The Connection Story)

**목표:** 서비스의 확장성을 보여주며, '개인'을 넘어 '관계(커플, 가족, 비즈니스 파트너)' 단위로 사용하도록 유도한다.
**포맷:** 시네마틱하고 감성적인 톤앤매너. (느린 음악, 부드러운 화면 전환).

| 요소 | 상세 내용 및 스크립트 가이드 | 실행 가능한 액션 아이템 |
| :--- | :--- | :--- |
| **훅(Hook)** | (0-3초) "가장 사랑하는 사람의 이름은 왜 다르게 불릴까요?" <br> *(시각: 두 손이 닿는 클로즈업, 따뜻한 색감)* | 보편적이고 감성적인 질문으로 공감을 유도. |
| **보이스오버** | (4-15초) "우리가 만나는 사람들의 이름은 각자의 서사를 지니고 있어요. 하지만 그 이름을 통해 우리는 '우리'라는 새로운 이야기를 만들어가죠." <br> (16-25초) "QARAH는 단지 나만의 이야기가 아니라, 당신의 관계에 가장 완벽하게 어울리는 서사까지 찾아드립니다." | 서비스가 개인을 넘어 관계 단위로 확장될 수 있음을 은연중에 강조. |
| **자막/화면 구성** | 🖼️ (이미지): 커플 명함 Mockup / 가족 이름 조합 예시<br> 🌟: '우리'를 완성하는 마지막 퍼즐 조각은? -> QARAH | 관계성(Relationship) 키워드를 중심으로 시각 자료 구성. |
| **캡션** | 💖 가장 소중한 사람과 나만의 서사를 만들어보세요. 이름에 담긴 의미는 개인의 것이지만, 그 가치는 '함께' 만들 때 비로소 완성됩니다. 사랑하는 사람에게 선물하고 싶은 QARAH 자산화 체험 링크! (태그: @친구계정) | 태그 기능을 활용하여 친구 간 바이럴을 유도합니다. |
| **해시태그** | #커플이름 #우리만의이야기 #서사공유 #관계의힘 #QARAH선물 #명언 | 감성적인 키워드와 관계 관련 해시태그를 사용. |
| **게시 시간** | 금요일 오후 7:30 (주말 계획 및 사적인 콘텐츠 소비가 활발한 저녁 시간) |
| **후속 스토리 아이디어** | "💍 태그한 친구/연인에게 QARAH 명함을 선물한다면, 어떤 서사를 담고 싶으신가요? 저희 전문가가 컨설팅해 드려요!" (직접 상담 유도) |
